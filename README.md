# Autohelm App

A cross-platform mobile application that emulates the Autohelm remote control for adjusting a vessel's Course Over Ground (COG) using NMEA0183 protocol over WiFi.

## Overview

This Flutter-based mobile app provides a digital interface matching the physical Autohelm remote control, allowing sailors to adjust their autopilot's course from their iOS or Android device via a Yacht Device WiFi Gateway.
